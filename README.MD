## 3.具体实现

1. 关于整数运算的优化：通过栈辅助，计算运算符号两侧规约
结果都是数字的情况。
2. 运算的实现：对于每个算式，规约后为其赋予一个新的临时
变量符号，并将该符号压入栈。
3. 变量名冲突检测：使用一个哈希表存储变量名和llv变量符
号的对应，在声明变量时检查表中是否有相同符号，若有，报错
若没有，进行变量的声明，若声明中包含赋值，则在声明分配内
存后将值存入指针对应地址。
